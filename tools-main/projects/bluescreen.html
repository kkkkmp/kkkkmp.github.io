Windows 11常见蓝屏代码参考表

错误代码 描述
0x0000000A IRQL_NOT_LESS_OR_EQUAL - 表示发生了内核或驱动程序的错误，可能导致系统崩溃。
0x0000001E KMODE_EXCEPTION_NOT_HANDLED - 表示内核模式发生异常，通常是由于驱动程序问题引起的。
0x0000003B SYSTEM_SERVICE_EXCEPTION - 表示系统服务遇到了问题，可能是由于驱动程序或硬件问题引起的。
0x0000004A IRQL_GT_ZERO_AT_SYSTEM_SERVICE - 表示内核或系统服务发生错误，可能导致系统崩溃。
0x00000050 PAGE_FAULT_IN_NONPAGED_AREA - 表示发生了内存错误，通常是由于硬件或驱动程序问题引起的。
0x0000007F UNEXPECTED_KERNEL_MODE_TRAP - 表示内核模式遇到了意外情况，通常由于硬件或驱动程序问题引起。
0x000000C2 BAD_POOL_CALLER - 表示发生了内存池错误，通常是由于驱动程序问题引起的。
0x000000EF CRITICAL_PROCESS_DIED - 表示关键进程已终止，可能由于系统文件损坏或硬件问题引起。
0x000000F4 CRITICAL_OBJECT_TERMINATION - 表示关键对象已终止，通常由于硬件或存储设备问题引起。
0x00000109 CRITICAL_STRUCTURE_CORRUPTION - 表示发生了关键结构损坏，可能由于驱动程序或硬件问题引起。
0x000001E3 RESOURCE_NOT_OWNED - 表示尝试释放未拥有的资源，通常由于驱动程序问题引起。
0x0000021A STATUS_SYSTEM_PROCESS_TERMINATED - 表示系统进程已终止，通常是由于系统文件损坏引起的。
0x0000007E SYSTEM_THREAD_EXCEPTION_NOT_HANDLED - 表示系统线程发生异常，通常由于驱动程序问题引起。
0x000000D1 DRIVER_IRQL_NOT_LESS_OR_EQUAL - 表示驱动程序引发的问题，可能导致系统不稳定。
0x00000133 DPC_WATCHDOG_VIOLATION - 表示延迟处理程序（DPC）监视超时，通常与驱动程序相关。
0x00000139 KERNEL_SECURITY_CHECK_FAILURE - 表示内核安全检查失败，可能由于内核数据损坏或恶意软件引起。
0x00000141 VIDEO_TDR_FAILURE - 表示显示驱动程序响应超时，通常与显卡或驱动程序相关。
0x00000124 WHEA_UNCORRECTABLE_ERROR - 表示硬件错误，可能与CPU、内存或其他硬件相关。
0x00000150 MEMORY_MANAGEMENT - 表示内存管理错误，可能由于内存模块或驱动程序问题引起。
